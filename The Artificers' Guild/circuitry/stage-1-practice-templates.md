# Stage I Practice Templates | 第一阶段 练习产物模板

The chamber does not ask first for elegance; it asks first for a trace that another eye can follow.

本殿首先要求的并非优雅，而是一条他人也能追下去的痕迹。

---

## Leaf Charge | 卷叶之责

Use this leaf beside [stage-1-beginner-tasks.md](stage-1-beginner-tasks.md). Template one answers task one, template two answers task two, and so on.

此卷应与 [stage-1-beginner-tasks.md](stage-1-beginner-tasks.md) 配合使用。模板一对应任务一，模板二对应任务二，余此类推。

## Template One | 模板一

### Component Note | 元件记录

- Part name 元件名称
- Symbol or common mark 符号或常见标记
- Nominal value or type 标称值或类别
- Handling caution 操作注意
- Expected use 预期用途
- Recorder's short note 记录短注

## Template Two | 模板二

### Visible Path Record | 可见路径记录

- Practice name 练习名称
- Power source 供电来源
- Declared path 已声明路径
- Measurement point one 测量点一
- Measurement point two 测量点二
- Observed output 观测输出
- Path match judgment 路径一致性裁定

## Template Three | 模板三

### Sensor Reading Record | 传感读数记录

- Sensor name 传感名称
- Baseline reading 基线读数
- Changed condition 改变条件
- New reading range 新读数范围
- Declared unit and drift 已声明单位与漂移
- First interpretation 首次解释

## Template Four | 模板四

### Broken Loop Trace | 断链追索记录

- Practice name 练习名称
- Intended path 预期路径
- First broken point 首个断裂点
- Measurement used 所用测量
- Suspected cause 可疑原因
- Next safe correction 下一步安全修正

## Template Five | 模板五

### First Handoff Note | 首份交接说明

- Signal or part name 信号或部件名称
- From hall 交出分殿
- To hall 接收分殿
- Declared range or trigger 已声明范围或触发条件
- Known limit 已知限制
- Required check before use 使用前必查项

## Closing Rule | 结尾规则

If the written signal still requires spoken repair, then the record is not yet ready to leave the bench.

若写下来的信号说明仍需要口头补修，这份记录便还不配离开工位。
