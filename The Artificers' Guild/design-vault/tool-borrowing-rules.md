# Tool Borrowing Rules | 器具借用规则

No bench remains orderly if its tools vanish into private memory.

若一间工位的器具会消失在私人记忆里，那间工位便不可能长久有序。

---

## Borrowing Charge | 借用之责

These rules govern how guild tools leave a rack, cross a hall, and return fit for the next pair of hands.

这些规则约束公会器具如何离架、跨殿流转，并以适合下一双手接用的状态归还。

## Borrowing Rules | 借用规则

1. Record the tool before removing it from its rack. 器具离架之前，先登记。
2. Declare the bench, construct, and intended task. 说明工位、构造名称与预定用途。
3. Inspect the tool before use and after return. 使用前后都要检查器具状态。
4. Do not relend a borrowed tool by word of mouth alone. 已借出的器具不得只靠口头再转借。
5. A damaged tool returns with a written note, not silent apology. 损坏器具归还时必须附带书面说明，而不是沉默致歉。
6. Any tool that no longer satisfies [safety-discipline.md](safety-discipline.md) must be withdrawn from use until cleared. 凡不再符合 [safety-discipline.md](safety-discipline.md) 要求的器具，必须先停用，再等待复核。

## Minimum Borrowing Ledger | 最低借用账目

- Tool name 器具名称
- Rack mark or storage place 架位标记或存放位置
- Borrower 借用者
- Hall and bench 分殿与工位
- Construct or task 构造或任务
- Borrowed date 借出日期
- Intended return date 预计归还日期
- Condition at borrowing and return 借出与归还时状态

## Cross-Hall Expectations | 跨殿借用约定

- Tools leaving circuitry must return free of loose wire, solder scraps, and live residue. 从 circuitry 借出的器具归还时，不得残留飞线、焊屑或带电风险。
- Tools leaving mechanica must return free of trapped filings, bent jaws, and hidden strain. 从 mechanica 借出的器具归还时，不得残留金属屑、变形夹口或隐性受力。
- Tools leaving golemcraft must return with adapters, cables, and configuration notes restored as found. 从 golemcraft 借出的器具归还时，转接件、线缆与配置说明都应恢复原状。

## Loss and Damage | 遗失与损伤

If a tool is lost, damaged, or found unfit for the next apprentice, the borrower records:

若器具遗失、损伤，或已不适合下一位学徒接用，借用者应记录：

- What failed or went missing 损伤或遗失内容
- During which task it occurred 发生于哪项任务
- Whether work has been halted 是否已停工
- What repair, replacement, or inspection has been requested 已申请何种修复、更换或检验

## Guild Saying | 公会短谚

The tool returned clearly is worth more than the tool returned quickly.

归还得清楚的器具，比归还得仓促的器具更有价值。
