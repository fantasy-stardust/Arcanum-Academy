# design-vault | 设计密库

This is the sealed chamber where scattered ingenuity is ordered before it is assembled.

这里是让零散巧思在装配之前先被整理归序的密室。

---

## Vault Charge | 密库之责

The Design Vault keeps the guild's shared conventions for naming, interfacing, assembly, apprentice readiness, tool discipline, and measurement practice. It is consulted whenever code, circuits, and mechanisms must work together without confusion.

设计密库保存公会共用的命名、接口、装配、学徒准备、器具规训与测量约定。凡是代码、电路与机构需要彼此协作而不致混乱时，都应先来此查阅。

## Vault Use | 使用方式

The Design Vault is not meant to replace hall instruction. Each hall keeps its own craft; the vault is only concerned with points of connection, handoff, and shared record.

设计密库并不取代各分殿内部的教学。各殿各自守其工艺，而密库只负责那些发生在连接、交接与共用记录处的规则。

## Vault Saying | 密库短谚

What is aligned in design need not be rescued in haste.

凡在设计中对齐之物，便不必在仓促中被补救。
