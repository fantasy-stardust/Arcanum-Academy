\section*{粗切与精切：整数除法与浮点除法 \\ (Integer vs Floating-point Division)}

\begin{lstlisting}
// 整数除法
int x = 7, y = 2;
cout << x / y << endl;  // 3（不是3.5）

// 浮点除法
// 方法1：C风格类型转换 (double)x / y
double r1 = (double)x / y;  // 3.5
// 方法2：使用浮点数字面量 x / 2.0
double r2 = x / 2.0;  // 3.5
// 方法3：C++风格类型转换 static_cast<double>(x) / y
double r3 = static_cast<double>(x) / y;  // 3.5
// 方法4：函数式类型转换 double(x) / y
double r4 = double(x) / y;  // 3.5
cout << r1 << " " << r2 << " " << r3 << " " << r4 << endl;
\end{lstlisting}

\section*{符文词典 (Arcane Glossary)}

\begin{center}
\begin{tabular}{|l|l|p{10cm}|}
\hline
\textbf{英文名词} & \textbf{中文名词} & \textbf{解释} \\
\hline
Integer Division & 整数除法 & 两个整数相除，结果舍去小数部分，只保留整数——如同铸灵体粗切材料，碎屑被直接丢弃。如 7 / 2 = 3。 \\
\hline
Floating-point Division & 浮点除法 & 至少一个操作数为浮点数，结果为带小数的精确值——精切模式，保留每一分材料。如 7.0 / 2 = 3.5。 \\
\hline
Type Casting & 类型转换 & 将一种符记类型转换为另一种，如 \texttt{(double)x} 将整数符记 x 转为浮点数。铸灵体借此在粗切与精切之间切换。 \\
\hline
Truncation & 截断 & 去掉小数部分，只保留整数部分。整数除法会向零截断，如同截去不完整的碎片。 \\
\hline
\end{tabular}
\end{center}
