# Stage I Practice Templates | 第一阶段 练习产物模板

The hall's first models are judged by whether force, fit, and motion can be described without waste.

本殿对起步模型的判断，在于受力、配合与运动能否被无浪费地写清。

---

## Leaf Charge | 卷叶之责

Use this leaf beside [stage-1-beginner-tasks.md](stage-1-beginner-tasks.md). Template one answers task one, template two answers task two, and so on.

此卷应与 [stage-1-beginner-tasks.md](stage-1-beginner-tasks.md) 配合使用。模板一对应任务一，模板二对应任务二，余此类推。

## Template One | 模板一

### Force Path Sketch Record | 力路草图记录

- Practice name 练习名称
- Object or mechanism 物件或机构
- Force entry point 受力进入点
- Resistance point 受阻点
- Intended stop point 预期止位
- Three short observations 三条简短观察

## Template Two | 模板二

### Fit Record | 配合记录

- Part or joint name 构件或接点名称
- Measured place 测量位置
- Baseline dimension 基线尺寸
- Acceptable range 可接受范围
- Declared unit 已声明单位
- Why the fit matters 配合为何重要

## Template Three | 模板三

### Transmission Prediction Note | 传动预测记录

- Practice name 练习名称
- Transmission form 传动形式
- Predicted change 预测变化
- Observed change 实测变化
- Difference between them 二者差异
- First explanation 首次解释

## Template Four | 模板四

### Linkage Motion Note | 连杆运动记录

- Practice name 练习名称
- Moving part 起动构件
- Receiving part 受动构件
- Travel or arc 行程或摆角
- Stop point 止位
- Likely strain point 可能受力偏差点

## Template Five | 模板五

### Balance and Wear Log | 平衡与磨损日志

- Practice name 练习名称
- Repeated motion type 重复运动类型
- Cycle count or duration 循环次数或持续时长
- First sign of imbalance or wear 首个失衡或磨损征兆
- Where it appeared 征兆出现位置
- Next maintenance or adjustment 下一步维护或调整

## Closing Rule | 结尾规则

If the note shows only that the model moved, but not how it strained, settled, or drifted, then the motion has not yet been properly observed.

若一份记录只说明模型动了，却没有写清它如何受力、归定或漂移，那么这段运动便还没有被真正观察。
