# Oscillating Gate Carriage | 摆振门车案例

## Record State | 入档状态

`failed-but-instructive`

## Hall Configuration | 分殿组合

golemcraft + circuitry + mechanica

## Function Domain | 功能类型

Gate opening, return motion, unstable feedback, compounded excess.

门体开启、回位运动、不稳定反馈与偏差叠加。

## Why It Is Kept | 何以留卷

The guild keeps this failure because it demonstrates a hard truth: when each hall is only slightly excessive, the finished construct may become violently excessive.

公会保留这份失败，是因为它说明了一个严酷事实：当每一座分殿都只“略微过量”时，最终构造却可能变得剧烈失控。

## Failure Marks | 失效特征

- Late feedback threshold 反馈阈值过晚
- Overlong control pulse 控制脉冲过长
- Counterweight beyond balance point 配重越过平衡点

## Proper Use | 正确用法

Read this entry before any apprentice attempts a build whose motion can strike a stop with increasing force.

凡学徒准备尝试可能以递增力量撞击止挡的运动构造前，都应先读此条目。
