# failure-cases | 失效案例

These records are kept so that a future apprentice need not pay twice for the same confusion.

这些记录之所以被保存，是为了让后来的学徒不必为同一种混乱付出两次代价。

---

## Recording Rule | 记录规则

Every failure record should state the intended construct, the first visible symptom, the traced cause, the correction applied, and the lesson kept.

每一份失效记录都应说明目标构造、最先出现的症状、追溯到的原因、采取的修正，以及最终留下的教训。
