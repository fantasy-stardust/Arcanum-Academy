# test-protocols | 受试规程

Each hall proves its work differently, but all proof must be explicit.

各分殿验证其成果的方式可以不同，但一切验证都必须清楚写明。

## Shared Expectation | 共用要求

Every protocol must state what is expected, how it is measured, what counts as failure, and what must be revised before retest.

每份规程都必须说明预期结果、测量方式、何种情况算作失败，以及在复试之前必须修订何处。
