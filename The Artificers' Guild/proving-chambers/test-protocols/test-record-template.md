# Test Record Template | 受试记录模板

Use this sheet whenever a construct enters the Proving Chambers.

凡构造进入试验间，都可使用此表进行记录。

---

## Construct Name | 构造名称

## Trial Date | 受试日期

## Participating Halls | 参与分殿

## Trial Goal | 受试目标

## Expected Result | 预期结果

## Observed Result | 实际结果

## Measurements | 测量数据

- Time / timing 时间 / 时序
- Signal / state 信号 / 状态
- Motion / load 运动 / 载荷

## First Failure Symptom | 首发失效症状

## Traced Cause | 追溯原因

## Required Revision | 必要修订

## Chamber Decision | 试验间裁定

`pass | revise-and-retest | archive-as-failure`

## Recorded By | 记录者

## Approved By | 批准者
