# Stage I Practice Templates | 第一阶段 练习产物模板

These templates are the smallest written forms the hall will accept for its first workings.

这些模板，是本殿对起步练习所接受的最小书写形态。

---

## Leaf Charge | 卷叶之责

Use this leaf beside [stage-1-beginner-tasks.md](stage-1-beginner-tasks.md). Template one answers task one, template two answers task two, and so on.

此卷应与 [stage-1-beginner-tasks.md](stage-1-beginner-tasks.md) 配合使用。模板一对应任务一，模板二对应任务二，余此类推。

## Template One | 模板一

### State Chain Record | 状态链记录

- Practice name 练习名称
- Intended routine 预期流程
- Named states 具名状态
- Transition trigger for each state 每个状态的转移触发
- First doubtful threshold 首个可疑阈值
- Recorder's after-note 记录后记

## Template Two | 模板二

### Repeatable Run Log | 可复现运行日志

- Practice name 练习名称
- Repeated action 重复动作
- Declared measurement and unit 已声明测量与单位
- Run one result 第一次结果
- Run two result 第二次结果
- Run three result 第三次结果
- Stable or unstable judgment 稳定或不稳定裁定

## Template Three | 模板三

### Fault Trace Note | 故障追索记录

- Practice name 练习名称
- Expected result 预期结果
- First visible symptom 首发症状
- Added trace or log points 补入的追索点或日志点
- Where expectation first broke 预期首次断裂之处
- Next correction to attempt 下一步修正尝试

## Template Four | 模板四

### Function Contract Note | 函数契约说明

- Function name 函数名称
- Accepted input 接收输入
- Returned output 返回输出
- Declared responsibility 已声明职责
- Safe-failure condition 安全失败条件
- One proving case 一条验证样例

## Template Five | 模板五

### Utility Test Record | 小工具测试记录

- Tool name 工具名称
- Tool charge 工具职责
- Case one and what it proves 样例一及其证明内容
- Case two and what it proves 样例二及其证明内容
- Case three and what it proves 样例三及其证明内容
- Present weakness 当前薄弱点

## Closing Rule | 结尾规则

If the record cannot be reread a day later with full meaning still intact, the practice has not yet been written clearly enough.

若隔了一日再读时，这份记录已不能保留完整意义，那么这次练习便还没有被写得足够清楚。
