# golemcraft | 信息技术

In this forge, thought is translated into instructions, systems, and tools that can truly run.

这里是把思想译作指令、系统与可运行工具的工坊。

---

## Hall Charge | 学科之责

Golemcraft studies computation, programming, data, networks, and digital systems. It teaches apprentices to break problems into parts, name things precisely, test what they build, and turn logic into dependable craft.

信息技术之学关注计算、编程、数据、网络与数字系统。它要求学徒学会拆解问题、准确命名、验证构造，并把逻辑锻造成值得信赖的工艺。

## What Is Studied Here | 研习内容

- Computational thinking and problem decomposition 计算思维与问题拆解
- Programming syntax, semantics, and debugging 编程语法、语义与调试
- Data structures, algorithms, and automation 数据结构、算法与自动化
- Operating environments, networks, and system design 运行环境、网络与系统设计

## Suggested Holdings | 建议收录

- Concept notes and command references 概念笔记与命令参考
- Code experiments and small utilities 代码实验与小型工具
- Project logs and version records 项目日志与版本记录
- Bug reports, debugging traces, and fixes 问题报告、排错轨迹与修复记录
- Architecture sketches and reusable snippets 结构草图与可复用片段

## Hall Saying | 分殿短谚
If thought can be named clearly enough, it can often be built.

若一种思想能被足够清晰地命名，它往往也就能被建造出来。

A tool worthy of trust is built twice: once in thought, once in testing.

可信之器往往会被建造两次：先在思想中，再在验证里。
