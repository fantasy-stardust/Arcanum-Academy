# golemcraft | 信息技术

This is the forge where logic is taught to take shape as language, structure, and working tool.

这里是让逻辑学会拥有形体的工坊，它最终会化为语言、结构与可运行的工具。

---

## Hall Charge | 学科之责

Golemcraft studies computation, programming, systems, digital tools, and the art of building with code. It values clarity, modularity, testing, and the discipline of making ideas executable.

信息技术之学关注计算、编程、系统、数字工具与以代码建造事物的工艺。它重视清晰、模块化、验证，以及把思想变成可执行结构的能力。

## What Is Studied Here | 研习内容

- Computational thinking 计算思维
- Programming languages and syntax 编程语言与语法
- Data, algorithms, and logic structures 数据、算法与逻辑结构
- Projects, tools, and system design 项目实践、工具使用与系统设计

## Suggested Holdings | 建议收录

- Concept notes 概念笔记
- Code experiments 代码实验
- Project logs 项目日志
- Bug records and fixes 问题记录与修复
- Reference snippets 参考片段

## Hall Saying | 分殿短谚

If thought can be named clearly enough, it can often be built.

若一种思想能被足够清晰地命名，它往往也就能被建造出来。
