# Self-Return Lantern Lift | 自回位灯架升降构造

## Record State | 入档状态

`planned`

## Hall Configuration | 分殿组合

golemcraft + circuitry + mechanica

## Function Domain | 功能类型

Timed lift, guided travel, held position, balanced return.

定时升降、导向行程、停驻保持与平衡回位。

## Why It Belongs Here | 何以入卷

This planned entry is a stronger test of alignment than the study-bench sentinel because timing drift and weight imbalance reveal themselves quickly and without mercy.

相较于学习台守望构造，这条待造条目对对位的考验更强，因为时序漂移与重量失衡都会迅速而无情地暴露出来。

## Build Watch | 构造警示

- Do not trust a hold interval that has not been written with a clear unit. 凡未以明确单位写出的停驻时长，都不值得信任。
- A counterweight that is only almost balanced is already a future failure. 只差一点点才平衡的配重，本身就已是未来失效。
- Rest-position feedback must settle before the cycle is called complete. 静止点反馈必须先稳定，循环才可宣告结束。

## Promotion Condition | 升格条件

This entry should not rise beyond `planned` until it can repeat its return point under more than one load condition.

唯有在不止一种载荷条件下都能重复回到同一静止点后，此条目才应脱离 `planned`。
