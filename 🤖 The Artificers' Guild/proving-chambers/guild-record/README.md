# guild-record | 公会档案

The guild record preserves what has been judged clear enough to teach from, reuse, or trust.

公会档案保存那些已经被判断为足够清楚、足够可靠，因而值得教学、复用或信赖的作品。

---

## Record States | 入档状态

- `current` 当前仍可作为有效参考
- `archived` 已归档保存，但不再作为首选范本
- `superseded` 已被更清楚或更可靠的版本取代
- `failed-but-instructive` 未通过，但仍具教学价值

## Rule of Entry | 入档规则

A construct enters the guild record only when its purpose, interfaces, proof, and revision history can all be read without relying on memory alone.

只有当构造物的用途、接口、验证与修订历史都能在不依赖口头记忆的情况下被读取时，它才可入档。
