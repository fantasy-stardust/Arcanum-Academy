# Self-Return Lantern Lift Brief | 自回位灯架升降构造简报

## Purpose | 用途

Create a compact lifting frame that raises a lantern on command, holds briefly, and returns to its starting height without manual correction.

构造一套紧凑的升降架，使其能够在接收指令后升起灯具、短暂停驻，并在无需人工校正的情况下回到初始高度。

## Hall Roles | 分殿分工

- golemcraft: command sequence and hold interval
- circuitry: drive trigger and stop feedback
- mechanica: frame guidance, lift path, and counterweight balance

## Minimum Success | 最低完成标准

- Lift remains aligned during travel 升降过程中保持对位
- Hold interval does not drift 停驻时长不漂移
- Return path ends at a repeatable rest point 回位路径能稳定落在同一静止点

## Suggested Proof Focus | 建议受试重点

- pulse duration 脉冲持续时间
- counterweight balance 配重平衡
- rest-position feedback 静止点反馈
