# Feedback Loops | 反馈回路

The guild treats feedback as the path by which motion returns to judgment.

公会将反馈视为让运动返回判断之处的路径。

---

- Measure something visible or repeatable 测量某种可见或可重复之物
- Return that measurement in a form the logic can interpret 以逻辑能够解释的形式回传测量结果
- Use feedback to revise thresholds, not to excuse bad records 用反馈修订阈值，而不是为糟糕记录开脱
- If feedback cannot be explained, the loop is not yet trustworthy 若反馈无法被清楚解释，这条回路就还不可信
